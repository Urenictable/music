# music

#### 介绍
一个基于Tkinter的操作界面
1.  本产品主要用于下载音乐
2.  注意：此产品因技术限制，目前只能下载可以完整听的音乐，若只能试听则该歌曲无法下载！


#### 软件架构
1.  软件架构说明
2.  主要以Python语言
3.  music.py为主程序
4.  packages/easys.py为方便使用而定义的库


#### 安装教程

1.  在cmd中输入 git clone https://gitee.com/cyz-h/musics.git 即可下载
2.  本程序用到的库有： os sys time json requests 
3.  注意：在本人定义的库中已有 第三方库自动补全 程序，无需手动下载该程序所需要的第三方库

#### 库引用说明

1.  os: 判断文件/文件夹是否存在 
2.  sys: 引用自定义的库
3.  time：设置时间间隔，避免网站反爬
4.  json: 解析请求网页或取得json数据
5.  requests: 请求网页，获取数据,下载音乐


#### 使用说明

1.  打开cmd终端
2.  输入cd Musics
3.  输入 python music.py 即可
